# Java中的集合特性
![集合层级图](/assets/jihe.png)

1. ArrayList：访问方便，增删不方便，如果要大量的访问，要用到ArrayList
2. LinkedList：增删方便，访问不方便，如果要大量的增删，要用到LinkedList
3. Stack：主要描述后进先出
4. Queue：主要描述先进先出
5. HashSet：主要为了实现去重
6. Map：相当于增加了一个目录
